# Finance Dashboard

A simple finance dashboard built using React and Tailwind CSS.

## Features:
- Track income and expenses
- Display balance summary
- Search/filter transactions
- Clean and responsive UI

## Tech Stack:
- React (Vite)
- Tailwind CSS
- JavaScript

## How to Run:
1. Install dependencies:
   npm install

2. Run project:
   npm run dev

3. Open in browser:
   http://localhost:5173
